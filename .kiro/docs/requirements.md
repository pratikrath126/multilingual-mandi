# Multilingual Mandi - PRD

## Overview
A real-time linguistic bridge for India's local agricultural markets (Mandis).

## Core Requirements
1.  **Linguistic Bridge:** Voice-to-voice and text-to-text translation between 22+ regional Indian languages and trade-standard Hindi/English.
2.  **Price Discovery:** Real-time integration with APMC/Market price feeds for transparent crop valuation.
3.  **Negotiation Support:** AI-mediated chat interface to facilitate fair trade agreements.
4.  **Accessibility:** High-contrast, mobile-first design with a voice-primary interaction model.

## User Persona
Local vendors and small-scale farmers who may have low literacy but high mobile usage.
